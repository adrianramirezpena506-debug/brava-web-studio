# Brava Web Studio

Sitio web premium en español para Brava Web Studio, desarrollado de forma independiente con HTML5, CSS3, JavaScript ES Modules y Vite.

## Desarrollo local

```bash
npm install
npm run capture:portfolio
npm run create:og
npm run dev
```

## Verificación

```bash
npm run lint
npm run build
```

El proyecto no contiene configuración de despliegue. Todos los CTA de contacto se generan desde una única variable de WhatsApp en `src/main.js`.
